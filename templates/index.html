What is BaybayAtin

BaybayAtin is an web application
made using HTML5, CSS, Javascript,
and Image Processing using Python
with Tensorflow and Keras. It is to
provide a tool that will help Filipinos
to have an understanding and
knowledge about the Philippine
Ancient Script, Baybayin.

To provide more information about
Baybayin to students and anybody else who

well-informed website that
